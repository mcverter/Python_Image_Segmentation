Contour Tracer¶

The contour tracer is responsible for determining a set of contours for an image.

class ContourTracer.ContourTracer¶

The contour tracer is responsible for determining a set of contours for an image.

Given a 2D array of points, it analyses these points and returns a set of contours for the image.

Attributes

image nparray Image being analyzed
mask nparray Image with defined contours
xnow, ynow int Indexes of pixel being analyzed
