Graph Cut Framework¶

A GraphCutFramework coordinates together (1) The pixel intensity observations taken from the original image (2) The object seeds given at startup (3) The background seeds given at startup (4) The estimation model for the source region (5) The estimation model for the sink region (6) The estimation model for the boundary between these two regions

class GraphCutFramework.GraphCutFramework(image_interface, estimator, crf_lambda)¶

A GraphCutFramework coordinates together (1) The pixel intensity observations taken from the original image (2) The object seeds given at startup (3) The background seeds given at startup (4) The estimation model for the source region (5) The estimation model for the sink region (6) The estimation model for the boundary between these two regions

The processing done by this framework is fed into the Boykov-Kolmogorov Graph Cut Energy Minimization algorithm in order to delineate target segments from non-target segments.

The GraphCutFraumework is an abstract class and should never be instantiated as an object Instead, one should use the MAP_CRF_Framework or the BoykovFramework
