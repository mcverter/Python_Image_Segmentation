class BoykovFramework.BoykovFramework(image_interface, estimator, crf_lambda, boykov_sigma)¶

The Boykov Framework inherits from the GraphCutFrameowork The Boykov_Framework coordinates together (1) The pixel intensity observations taken from the original image (2) The object seeds given at startup (3) The background seeds given at startup (4) The estimation model for the source region (5) The estimation model for the sink region (6) The estimation model for the boundary between these two regions

The processing done by this framework is fed into the Boykov-Kolmogorov Graph Cut Energy Minimization algorithm in order to delineate target segments from non-target segments.

The BoykovFramework is differentiated from the MAP_CRF _Framework in the following ways: (1) Differing ways of calculating the neighbor-edge weight (2) The Boykov Framework makes use of the boykov_sigma (3) The MAP_CRF_Framework makes use of a Conditional Random Field recalculation of the Maximum A Priori.
